Key structural decisions in this layout

  • Branch bar spans full width — sits between TopBar and the main flex row (nav rail + nav tree + content)
  • Nav tree shows viewing branch only — signal dots (orange = overlay activity, blue = second branch colour) indicate nodes with incoming changes
  • Ghost rows render inline in SchemaCard — left stripe colour identifies which overlay branch the field comes from; conflicts get red stripe + ⚠
  • Branch bar hidden when no source is configured (filesystem-only mode) — no multi-branch capability without a git source
